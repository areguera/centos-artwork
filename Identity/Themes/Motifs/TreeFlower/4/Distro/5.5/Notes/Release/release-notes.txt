CentOS 5.5 Release Notes

   The CentOS project welcomes you to CentOS-5.5.

   The complete release notes for CentOS-5.5 can be found online at:
   http://wiki.centos.org/Manuals/ReleaseNotes/5.5/.

   A list of frequently asked questions and answers about CentOS-5.5
   can be found online at: http://wiki.centos.org/FAQ/5.5/.

   If you are looking for help with CentOS, we recommend you start at
   the  http://wiki.centos.org/GettingHelp/ for pointers to the
   different sources where you can get help.

   If  you  would like to contribute to the CentOS Project, see
   http://www.centos.org/ for areas where you could help.

   For more information about The CentOS Project in general please
   visit our homepage at: http://www.centos.org/.
